# Wildfire and Extreme Rainfall Radionuclide Remobilization Package

This folder contains a GeoMine Research paper package for the topic:

> Wildfire and extreme-rainfall controls on radionuclide remobilization in northern Canadian uranium-mining environments.

## Main Deliverables

- `Wildfire_Extreme_Rainfall_Radionuclide_Remobilization_Athabasca.zh.md`  
  Chinese academic paper draft with research-direction argument, data analysis, equations, figures, limitations, and conclusions.
- `Wildfire_Postfire_Water_Chemistry_Long_Term_Risk_Phase2.zh.md`  
  Phase-2 integrated academic paper that links public screening, first-flush water chemistry and lake/wetland sediment secondary sources into one causal prediction workflow.
- `figures/*.svg`  
  Reproducible scientific figures. Figures 1-7 are first-stage screening outputs; Figures 8-13 are phase-2 causal, water-chemistry, partitioning and long-term-risk outputs.
- `data/raw/`  
  Downloaded open-source data used in this run.
- `data/processed/`  
  Processed CSV/JSON tables generated by the script.
- `models/phreeqc_wildfire_ash_stream_mixing_template.phr`  
  PHREEQC input template for wildfire ash leachate and stream-water mixing. It is not an executed or calibrated model.
- `models/phreeqc_postfire_water_sensitivity_llnl.phr`  
  Executed PHREEQC sensitivity input anchored to EARMP 2023 surface-water screen values and placeholder major-ion chemistry.
- `models/phreeqc_run_manifest_phase2.md`  
  PHREEQC run manifest, command, warnings and modeling boundary.
- `scripts/analyze_wildfire_rainfall_radionuclide.py`  
  Reproducible processing and figure-generation script.
- `scripts/analyze_phase2_water_sediment_risk.py`  
  Reproducible second-stage processing script for EARMP chemistry, open-data readiness, partitioning and long-term sediment-source sensitivity.
- `mcp_provenance.md`  
  GeoMine MCP and data-source provenance summary.

## Reproduce

From the repository root:

```bash
python3 report/2026-05-14-wildfire-rainfall-radionuclide-remobilization/scripts/analyze_wildfire_rainfall_radionuclide.py
```

Expected summary:

```json
{
  "facility_count": 5,
  "nbac_fire_feature_count": 1362,
  "nbac_year_min": 1972,
  "nbac_year_max": 2024,
  "key_lake_daily_records_after_merge": 1827,
  "wsc_daily_records": 1827,
  "hydrometric_station_count": 10,
  "release_record_count": 300,
  "top_screening_facility": "Rabbit Lake",
  "top_screening_index": 0.456
}
```

Run the phase-2 integrated analysis:

```bash
python3 report/2026-05-14-wildfire-rainfall-radionuclide-remobilization/scripts/analyze_phase2_water_sediment_risk.py
```

Expected phase-2 summary:

```json
{
  "earmp_total_records": 6680,
  "earmp_surface_water_records": 511,
  "open_data_layers": 11,
  "downloaded_or_parsed_layers": 5,
  "metadata_or_catalog_layers": 4,
  "critical_gaps": [
    "Fire burn severity",
    "Lake/wetland sediment cores"
  ]
}
```

## Boundary

The screening index is for research prioritization and sampling design only. It is not a dose estimate, regulatory risk finding, environmental compliance conclusion, Qualified Person opinion, or mine safety conclusion.
