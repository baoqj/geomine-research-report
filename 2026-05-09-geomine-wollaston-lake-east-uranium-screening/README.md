# GeoMine Research Package: Wollaston Lake East Uranium Screening

This directory contains a GeoMine evidence-lane screening package for the Athabasca Basin margin uranium target area normalized as Wollaston Lake East / eastern Wollaston area, Saskatchewan, Canada.

Files:

- `wollaston_lake_east_uranium_screening_report.md` - full screening memo with evidence matrix, assumptions, data gaps, and next steps.
- `evidence_matrix.csv` - compact evidence matrix for reuse.
- `machine_readable_summary.json` - structured JSON summary.
- `mcp_and_data_provenance.md` - GeoMine MCP calls, ArcGIS queries, CDoGS source notes, and local-data checks.

Research date: 2026-05-09
