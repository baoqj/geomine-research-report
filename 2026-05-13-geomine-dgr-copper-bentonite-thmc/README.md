# GeoMine DGR Copper-Bentonite-Crystalline Rock THMC Study

This folder contains a GeoMine Research / THMC Modeling package for the near-field system of a crystalline-rock deep geological repository.

- `DGR_Copper_Bentonite_Crystalline_Rock_THMC_Model.zh.md` - main Chinese academic modeling report.
- `mcp_provenance.md` - MCP provenance, warnings, and data-boundary notes.
- `scripts/generate_thmc_screening_figures.py` - reproducible screening calculations and figure generation.
- `data/thmc_parameter_sources.csv` - source-to-parameter provenance table.
- `data/thmc_screening_results.csv` and `data/thmc_screening_summary.json` - machine-readable calculation outputs.
- `figures/fig1_nearfield_concept.svg` through `figures/fig8_scenario_matrix.svg` - paper figures inserted into the Markdown/PDF.

Status: literature-grounded conceptual model plus reproducible screening calculations and figures. It is not a calibrated safety assessment, regulatory decision, engineering certification, or site-specific performance assessment.
