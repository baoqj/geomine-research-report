# GeoMine Fractured-Rock Radionuclide Sorption / Diffusion / Colloid Study

This folder contains a GeoMine Research / THMC Modeling package for radionuclide migration in crystalline-rock fractures.

- `Fractured_Rock_Radionuclide_Sorption_Diffusion_Colloid_THMC_Model.zh.md` - main Chinese academic modeling paper.
- `Fractured_Rock_Radionuclide_Sorption_Diffusion_Colloid_THMC_Model.zh.pdf` - exported PDF.
- `mcp_provenance.md` - MCP provenance, warnings, and data-boundary notes.
- `scripts/generate_fracture_colloid_figures.py` - reproducible screening calculations and SVG generation.
- `data/fracture_colloid_parameter_sources.csv` - source-to-parameter provenance table.
- `data/fracture_colloid_nuclide_parameters.csv` - radionuclide grouping and screening parameters.
- `data/fracture_colloid_screening_results.csv` and `data/fracture_colloid_screening_summary.json` - machine-readable calculation outputs.
- `figures/fig1_fractured_rock_concept.svg` through `figures/fig8_scenario_risk_matrix.svg` - paper figures inserted into the Markdown/PDF.

Status: literature-grounded conceptual model plus reproducible screening calculations and figures. It is not a calibrated safety assessment, regulatory decision, engineering certification, or site-specific performance assessment.
