# GeoMine Research Package: Cigar Lake Uranium Screening

This directory contains a GeoMine evidence-lane screening package for the Cigar Lake uranium mine area in northern Saskatchewan.

Files:

- `cigar_lake_uranium_screening_report.md` - full screening memo with evidence, assumptions, data gaps, and next steps.
- `machine_readable_summary.json` - structured JSON summary for reuse in downstream tools.
- `mcp_provenance.md` - GeoMine MCP tool calls, outputs, and limitations from this run.

Research date: 2026-05-08
