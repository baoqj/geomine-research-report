# GeoMine Tailings Seepage THMC Groundwater Paper

This folder contains a GeoMine Research academic-paper draft for sulfide-bearing tailings seepage into shallow groundwater.

- `Tailings_Seepage_THMC_Groundwater_Paper.zh.md` - main Chinese academic paper with THMC model formulation, derivations, evidence matrix, parameter tables, and figure plan.
- `mcp_provenance.md` - GeoMine MCP tool provenance and data-source boundary notes.

Status: conceptual academic modeling paper, not a calibrated site simulation.
